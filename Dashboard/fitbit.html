Connect your Fitbit.

Uses Google Fit API to read your Fitbit data (Fitbit syncs automatically to Google Fit on your phone). Requires a free Google Cloud project — takes about 5 minutes to set up.
  1. Go to console.cloud.google.com → create or select a project
  2. Search for Fitness API → click Enable
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Choose Web application — add the redirect URL shown below to Authorized redirect URIs
  5. Copy your Client ID and Client Secret
  6. In Netlify: add env vars GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, GOOGLE_REDIRECT_URI → redeploy
  7. Paste your Client ID and proxy URL below
